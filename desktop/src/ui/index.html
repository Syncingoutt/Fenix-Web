Welcome to Fenix
Initial Setup Required
Please select the UE_Game.log file from your Torchlight Infinite installation.

This file is typically located at:
[Steam Library]\\steamapps\\common\\Torchlight Infinite\\UE_game\\TorchLight\\Saved\\Logs\\UE_Game.log
Enable Cloud Sync?
Optional, but recommended
Enable syncing to share price updates and receive community updates across devices. We only read game log data (item IDs, prices, listing counts, timestamps) and never collect personal data. You can change this later in Settings.
Keeping sync off means no cloud price updates and no contribution to shared prices.
Disable Cloud Sync?
This will stop all cloud synchronization
Are you sure you want to disable Cloud Sync? You will stop receiving cloud price updates and will need to update prices manually.

Your local prices will remain unchanged, but you won't benefit from community price updates.
You can re-enable Cloud Sync anytime in Settings.
Include Compasses & Beacons?
Do you want to include compasses and beacons in your hourly tracking?

If yes, you'll be able to select which ones to include in the calculation.
Select Compasses & Beacons
Clear All History?
This action cannot be undone
Are you sure you want to delete all historical data? This will permanently remove:
  • All date entries
  • All sessions and earnings
  • Inventory snapshots
Your current inventory and settings will not be affected.
Delete Selected Sessions?
This action cannot be undone
Are you sure you want to delete 0 selected session(s)?
This will permanently remove the selected session data and inventory snapshots.
