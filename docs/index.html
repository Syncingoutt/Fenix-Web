Enable Cloud Sync?
Optional, but recommended
Enable syncing to share price updates and receive community updates across devices. We only read game log data (item IDs, prices, listing counts, timestamps) and never collect personal data. You can change this later in Settings.
Keeping sync off means no cloud price updates and no contribution to shared prices.
Disable Cloud Sync?
This will stop all cloud synchronization
Are you sure you want to disable Cloud Sync? You will stop receiving cloud price updates and will need to update prices manually.

Your local prices will remain unchanged, but you won't benefit from community price updates.
You can re-enable Cloud Sync anytime in Settings.
